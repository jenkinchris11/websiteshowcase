# Website Showcase

A Vue 3 + Vite portfolio that embeds live previews of completed client sites. Optimized for Netlify deployment.

## Scripts
- `npm install` – install dependencies
- `npm run dev` – start the local dev server
- `npm run build` – create a production build
- `npm run preview` – preview the production build locally

## Deployment
Netlify is configured via `netlify.toml` to build with `npm run build` and publish the `dist` directory.
